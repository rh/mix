Mix

Mix is a powerful command-line XML editor. Some of its features are:

  • XPath and XPath templates
  • Regular expressions
  • Batch editing

See what you can do with Mix:

  • Add elements, attributes and fragments

    Add a new column to a table:

    $ mix add-element xpath://table/tr name:td value:"Lorem ipsum"

  • Convert elements and attributes

    Convert an attribute to an element:

    $ mix convert-to-element xpath:/people/person/@name

  • Unwrap elements

    Unwrap all strong elements inside p elements:

    $ mix unwrap xpath://p/strong

  • Set values

    Set a simple value (this will replace the existing value):

    $ mix set xpath://table/tr value:"Lorem ipsum"

    Set all id attributes to a new GUID value:

    $ mix set-guid xpath://@id

  • Remove elements, attributes and comments

    Remove all inline styles:

    $ mix remove xpath://@style

    Remove a particular table:

    $ mix remove xpath://table[@id='foo']

    Remove all comments:

    $ mix remove xpath://comment()

  • Rename elements and attributes

    Rename all p's to div's:

    $ mix rename xpath://p name:div
